This database contain hospital map information from Gaode Map Platform 
https://lbs.amap.com, downloaded and cleaned by author.

Each file contains hospital from each province. File name is adcode of that province. Note: There are duplicate files ex: 250000 and 26000 are same. 

Please use adcode from AMap_adcode_citycode.xlsx as index to find data of particular province. Details in AMap_adcode_citycode.xlsx.
